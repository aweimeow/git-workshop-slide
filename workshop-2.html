GIT 原理運作

git init

當我們執行 git init ...
    • 會有一個 .git 資料夾
    • 資料夾中有：HEAD、index、objects、refs
      • HEAD: 指向目前分支
      • index: 暫存區資訊
      • objects: 所有資料都在這
      • ref: 提交物件的指標
